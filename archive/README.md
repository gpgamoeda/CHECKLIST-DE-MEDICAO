Esta pasta guarda os snapshots das versões anteriores do checklist, mantidos
apenas como **referência histórica**. Eles não fazem parte do app vigente e
**não são publicados** (o build só empacota o `index.html` da raiz).

| Arquivo               | Origem                                         |
| --------------------- | ---------------------------------------------- |
| `checklist-v1.html`   | Antiga `index.html` (primeira versão)          |
| `checklist-v2.html`   | v2                                             |
| `checklist-v3.html`   | v3                                             |
| `checklist-v4.html`   | v4                                             |
| `checklist-v4.1.html` | v4.1                                           |

A versão vigente (antiga "v5") foi promovida para o `index.html` na raiz do
repositório durante a Sprint 0. Veja `../CHANGELOG.md`.

> A partir daqui, **não crie novas cópias numeradas** (`checklist-v6.html` etc.).
> O versionamento passa a ser feito por Git + `CHANGELOG.md`. Esta pasta tende a
> ser removida quando o histórico não for mais útil — ele continua preservado no
> Git de qualquer forma.
